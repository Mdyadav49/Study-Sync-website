# 📘 StudySync Learning

**StudySync** is a modern, responsive educational landing page built using **HTML**, **CSS**, and **Font Awesome**.
It promotes an online learning platform that highlights features, testimonials, newsletter sign-up, and a clean, accessible layout for users of all devices.

---

## 🌟 Features

-  Responsive Navigation Bar with Mobile Menu Icon
-  Hero Section with Call-to-Actions
-  Company Trust Logos Section
-  Features Grid with Icons
-  Testimonials Section
-  Newsletter Sign-up Form
-  Footer with Social Links and Navigation

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Grid)
- Font Awesome (Icons)
- Google Fonts (Inter)
